# Labeling Schema

- **FR**: Functional Requirement (e.g., "System shall allow user login.")
- **NFR**: Non-Functional Requirement (e.g., "System shall respond within 2 seconds.")
- **CONSTRAINT**: Business/technical constraint (e.g., "Must run on Windows 11.")
- **IRRELEVANT**: Not a requirement (e.g., greetings, off-topic text).
- **AMBIGUOUS**: Needs clarification from stakeholder.
